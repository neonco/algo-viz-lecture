# algo-viz-lecture

Лекция для учителей «Визуализация алгоритмов: от сортировок к графам» (2 часа, онлайн, 2026).

**Сайт лекции:** https://neonco.github.io/algo-viz-lecture/

## Файлы

- `index.html` — лекция-одностраничник (деплоится целиком)
- `visualizer_template.html` — шаблон визуализатора в стиле «Под капотом float»
- `sorting_visualizer.html` — пузырьковая и шейкерная сортировки, свой массив, код на Python
- `recursion_tree_visualizer.html` — дерево рекурсии: Фибоначчи и факториал (задача 16 ЕГЭ)
- `turtle_visualizer.html` — черепашья графика (задача 6 ЕГЭ), точки внутри фигуры
- `bad_sorting_visualizer.html`, `bad_recursion_visualizer.html` — антипримеры для сравнения «плохой vs хороший»
